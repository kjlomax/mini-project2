CSS Snippet Cheat Sheet

Here is the exact syntax for commonly used css code. Click on any snipet of code and it will be selected so you can copy it.

Flexbox Row

To create a flexbox row layout, use these properties.

                    
                        .row {
                            display:flex;
                            flex-direction: row;
                            flex-wrap: wrap;}

Flexbox Column

Use these properties to create a flexbox column.

                    
                        .column {
                            display:flex;
                            flex-direction: column;}

CSS Grid Layout

To build a 12 column grid layout:

                    
                        .grid {
                            display: grid;
                            width: 100%;
                            grid-template-columns: repeat(12, 1fr);
                        }

Linear Gradients

This creates a background linear gradient from top to bottom.

                    
                        .linear-gradient-background {
                            background-image: linear gradient (
                                rgba(203, 19, 178, 0.8) 0%,
                                rgba(203, 19, 178, 0.8) 100%
                            );
                        }
