CSS Snippet Cheat Sheet

Here is the exact syntax for commonly used css code. Click on any snipet of code and it will be selected so you can copy it.

Flexbox Row

To create a flexbox row layout, use these properties.

                    .row {
                            display:flex;
                            flex-direction: row;
                            flex-wrap: wrap;}

Flexbox Column

Use these properties to create a flexbox column.

                    .column {
                            display:flex;
                            flex-direction: column;}

CSS Grid Layout

To build a 12 column grid layout:

                    .grid {
                            display: grid;
                            width: 100%;
                            grid-template-columns: repeat(12, 1fr);
                        }

Linear Gradients

This creates a background linear gradient from top to bottom.

                    .linear-gradient-background {
                            background-image: linear-gradient (
                                color1, color2
                            );
                        }

Box Transition Glow

Use transition and Box Shadows to glow on hover.

                    .card-header {
                        border-radius: 8px;
    transition:all 0.5s ease-in-out;}
                        
                        .card:hover,
                        .card:hover .card-header {
                            box-shadow: inset 0px 0px 8px green , 0px 0px 15px green  ;
                        }

Overlay Card with Title

Use position properties and negative margins to raise elements higher than their natural starting point.

                    .card-header {
                        position: relative;
                        margin-top: -20px;
                    }
